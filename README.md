# 프로젝트 소개
계획을 세우고 포스팅하며 한 화면에 유튜브를 보며 포스트를 작성할 수 있게끔 하는 애플리케이션으로 개인적으로 사용할 수 있게끔 구성하고 Jlaner인 이유는 기존 Planer에서 P가 MBTI에 증흑적인 성향인 반면 J는 계획적인 성향인 것에 착안해 P를 J로 바꿔 Jlaner로 했다.

개발 기간: 2024.07.27 ~ 2024.09.12 (6주)
